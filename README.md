## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.


Needs file .env.local
my version is:
```
HOSTNAME=localhost
PORT=3000
FETCH_URL=http://$HOSTNAME:$PORT
```
